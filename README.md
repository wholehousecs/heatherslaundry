# Heather's Laundry Spa 🧺✨

A modern, colorful website for Heather's Laundry Spa - Professional laundry and cleaning services in Miami.

## Features

- 🌈 Creative coral & teal design
- 📱 Fully responsive
- 🚀 Built with Next.js
- 📦 Services showcase
- 📅 Online booking integration
- 💬 Contact form
- 📝 Blog with laundry & cleaning tips
- ⭐ Customer reviews

## Services

- Laundry Services (wash & fold)
- Pick-up & Delivery
- Residential/Airbnb/Turnover Cleaning

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build & Deploy

```bash
npm run build
npm run export
```

Files will be in the `out` directory ready for GitHub Pages.
